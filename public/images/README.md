# Images Folder

Place your service images here with the following naming convention:

## Current Images Needed:
1. `shade1.jpg` - Car shade with curved design (first image you provided)
2. `gazebo1.jpg` - Luxury outdoor gazebo with stone pillars (second image)
3. `pagola1.jpg` - Modern wooden pagola entrance cover (third image)
4. `shade2.jpg` - Industrial cantilever shade structure (fourth image)
5. `gate1.jpg` - Automated sliding gate (add if you have)

## How to Add Images:
1. Save your images to this folder (public/images/)
2. Name them according to the convention above
3. Supported formats: .jpg, .jpeg, .png, .webp
4. Recommended size: 800x800px or larger (square aspect ratio works best)

## Using Images in Admin:
When adding/editing services in the admin panel, use the path:
- `/images/shade1.jpg`
- `/images/gazebo1.jpg`
- `/images/pagola1.jpg`
- etc.

The website will automatically display these images in the service cards.
