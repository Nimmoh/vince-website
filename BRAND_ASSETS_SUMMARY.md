## 🎯 Quick Usage Guide

### In Your Website Header
```tsx

<Image 
  src="/logo.svg" 
  alt="Vincevic Shades" 
  width={200} 
  height={60}
/>
```

### For Social Media Profile
Use `logo-square.svg` (400x400) - perfect for:
- Facebook profile picture
- Instagram profile
- Twitter profile
- LinkedIn company logo

### For Business Cards
Use `logo-light.svg` on white background
- Recommended size: 2-3 inches wide
- Export at 300 DPI for printing

### For Email Signature

```html
<img src="https://yoursite.com/logo.svg" 
     alt="Vincevic Shades" 
     width="150" 
     height="45">
```

## 📊 File Formats

All assets are in **SVG format** (Scalable Vector Graphics):
- ✅ Scales to any size without quality loss
- ✅ Small file size (fast loading)
- ✅ Works on all modern browsers
- ✅ Perfect for web and print
- ✅ Easy to edit if needed
